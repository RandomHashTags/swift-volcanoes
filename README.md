# swift-volcanoes
Volcanoes and other relevant information expanded on [SwiftSovereignStates](https://github.com/RandomHashTags/swift-sovereign-states), ready to be used offline.

## Requirements
- Swift >= 5.5
- iOS >= 15.0
- macOS >= 12.0
- tvOS >= 15.0
- watchOS >= 8.0
